Simple game. Three ways to play.

## 1) Solo Journaling

Best for writing-focused play.

- Read each objective prompt.
- Choose a pool and roll.
- Write 2 to 6 lines per objective.
- Track Battery, Stability, Successes, Glitches.
- Keep the pace soft and reflective.

Use this mode if you want emotional depth and quiet pacing.

## 2) Solo With Terminal

Best for fast mechanical play.

- Use the terminal app for commands and rolls.
- Follow prompts in order.
- Let the system track progression.
- Export transcript when done.

Use this mode if you want speed, structure, and fewer manual steps.
Access the terminal from the itch game page: https://mama666.itch.io/31st-of-february 

## 3) Single Player With a DM

Best for guided scenes.

- One player is the Candidate.
- One DM frames scenes and NPC behavior.
- DM can read objective prompts aloud.
- Candidate chooses rolls and outcomes.
- DM interprets glitches and success flavor in fiction.

Use this mode if you want dialogue, improvisation, collaborative storytelling and dramatic pressure.

## What Never Changes

Nothing, anything can change if you want.


## Safety and Tone

Suggested tools: https://ttrpgsafetytoolkit.com/ 

This game includes burnout themes, social pressure, and identity strain. Play with care.
