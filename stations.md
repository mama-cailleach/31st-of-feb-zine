Each loop has 4 stations.
Each station has fixed objectives.

## 1) Awakening

Theme: start state, body state, morning script.

Objectives:

- D: Bed
- O: Mirror
- W: Fuel

## 2) Journey

Theme: transit pressure, social friction, environment load.

Objectives:

- G: Connection
- L: Interaction
- Y: Environment

## 3) Interview

Theme: formal evaluation, role performance, contradiction.

Objectives:

- S: Situation
- T: Task
- A: Action
- R: Result (special check)

## 4) Return

Theme: aftermath, depletion, recovery cycle.

Objectives:

- B: Commute Back (special check)
- U: Recharge (special check)
- M: Receipt Printing (special check)

## Fast Use At Table

For each objective:

- Read prompt
- Resolve roll/check
- Apply changes
- Move on

For each station:

- Keep pool choices intentional
- Track resource drift
- Use station tone to color narration

## End of Station Notes

- After Interview, Result can sharply change Stability.
- During Return, Battery and loop continuity become the focus.
